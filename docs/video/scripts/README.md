# 台本（scripts）フォルダ

各コンテンツ（モジュール・レッスン）の動画台本を格納します。

## 使い方

1. **新規台本を作る**: `_template.md` をコピーし、ファイル名を `[モジュール]_lesson[番号]_[テーマ].md` 形式で付ける。
2. **モジュール別に整理する**: 必要に応じて `querysync/`、`infraplay/` などのサブフォルダを作成して整理する。
3. **作業手順**: 台本 → VOICEVOX で音声 → CapCut で編集、の流れは [../WORKFLOW.md](../WORKFLOW.md) を参照。

## ファイル一覧（例）

| ファイル | モジュール | 説明 |
|:--|:--|:--|

| [querysync/lesson01_select-basic.md](./querysync/lesson01_select-basic.md) | QuerySync | SELECT文の基本（サンプル） |
