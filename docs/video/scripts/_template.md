# 台本テンプレート

新規レッスン動画の台本を作るときは、このファイルをコピーして使ってください。  
ファイル名例: `[モジュール名]_lesson[番号]_[テーマ].md`（例: `querysync_lesson01_select-basic.md`）

---

## メタ情報

| 項目 | 内容 |
|:--|:--|
| モジュール | （例: QuerySync） |
| レッスンID | （例: qs-lesson-1） |
| 動画タイトル（案） | （例: SQL超入門！SELECTでデータを取ってみよう【めたん×ずんだもん】） |
| 想定尺 | （例: 約10分） |
| ステップ数 | 3〜7 のいずれか |

---

## タイムスタンプ一覧（チャプター用）

Movidrill の `videoTimestamp` は**秒数**でレッスンJSONに記録する。編集後に実尺で更新すること。

| 表示用（目安） | 秒（編集後に記入） | 内容 |
|:--|:--|:--|
| 00:00 | 0 | イントロ（学習ゴール） |
| （ステップごとに追加） | | |
| （まとめ） | | まとめ |

---

## 台本本文

- **話者**: **めたん**＝講師役（解説・キュー）、**ずんだもん**＝生徒役（反応・質問・共感）
- **（一拍）**: 短い間
- **（ポーズ）**: やや長い間
- **【演習キュー】** は Movidrill で止めるタイミングなので文言を変えないこと（めたんが読む想定）

---

### [00:00] イントロ

（めたん）

>
> （例）みんな、こんにちは。めたんだよ。今日は〇〇を一緒に学んでいくね。
> 最後まで見たら、〇〇ができるようになってるから、楽しみにしてて。

（ずんだもん）

>
> ずんだもんも一緒に勉強するんだ！ よろしくね。（一拍）

（めたん）

>
> じゃあ、始めよう。

---

### [01:30] Step 1: （ステップタイトル）

（めたん）

>
> （ここに Step 1 の説明を書く。一文は短めに。）
> （重要な用語は《》で囲むとテロップ用に分かりやすい。）
> 《〇〇》は、最初はちょっと難しく感じるかもしれないけど、大丈夫。一緒にやっていこう。

（ずんだもん）

>
> （任意）うん、ちょっと不安だけど、やってみるね。

（めたん）

>
> **【演習キュー】ここで手を動かしてみましょう！**
>
> （一拍）できたら、動画を再開してね。答えはこのあとすぐ見せるから。

---

### [03:45] Step 1 の答え・解説

（めたん）

>
> （解答の説明。次のステップに進む前に「正解はこうだよ」と見せる。）

（ずんだもん）

>
> （任意）なるほど、そういうことか！

---

### （以下、Step 2 〜 同様に繰り返し）

（必要に応じて Step 2, 3, ... を追加。1動画あたり 3〜7 ステップに収める。解説＝めたん、反応＝ずんだもんの役割を維持する。）

---

### [XX:XX] まとめ

（めたん）

>
> （今日やったことの一言まとめ。）次は〇〇をやっていくから、楽しみにしててね。

（ずんだもん）

>
> また次の動画で一緒に学ぼう！ またねー。

---

## 編集メモ（VOICEVOX / CapCut 用）

- **めたん**: ノート（感情）＝解説は落ち着きめ、まとめは少し明るめ
- **ずんだもん**: 生徒役なので「分かった」「不安」「やってみる」などで感情を変える
- テロップで強調する用語: 《》で囲んだ語
- BGM: 小さめ、技術解説向け

---

## レッスンJSON用 videoTimestamp（編集後に記入）

```json
"steps": [
  { "id": 1, "videoTimestamp": 0, ... },
  { "id": 2, "videoTimestamp": 90, ... }
]
```

※ 実際の秒数は CapCut で確定したあと、この台本とレッスンJSONの両方を更新すること。
